# HINATA Go 
## HINATA Go 是什么？

HINATA Go 是一款移动应用程序，可将您的手机变身为多功能街机游戏刷卡器，主要有以下功能：
* 通过 NFC 或外接 HINATA 读卡器来扫描卡片
* 查看卡片信息，支持所有能在游戏中使用的卡片
* 作为读卡器用于登录游戏，支持使用 SEGA 游戏， Konami 游戏， KanadeDX
* 提供了一个卡包功能，可以保存你的卡片信息
* 在支持的平台上连接实体 HINATA 读卡器，并进行设置修改或固件更新： [管理 HINATA 读卡器](/configuration)
