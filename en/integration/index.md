# HINATA Ecosystem

HINATA has access to the powerful PN532 and Rust ecosystems, making it easy to use the card reader in your own projects.

## PN532 Ecosystem

<HCard title="🛠️ PN532 Passthrough" link="pn532/" cols="1">
  HINATA can be used directly as a standard PN532 with third-party host software.
</HCard>

## Rust HINATA Library

<HCard title="🦀 libhinata-rs" link="rustlib/" cols="1">
  A Rust library for communicating with HINATA and HINATA Lite.
</HCard>
