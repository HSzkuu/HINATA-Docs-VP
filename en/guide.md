# HINATA Manual Contents

HINATA is a card reader that can be used not only with arcade games, but also with a rich ecosystem.  
HINATA provides many original features, such as Kanade DX connectivity and high-speed card reading, and works plug-and-play.  
The HINATA ecosystem is continuously expanding.

## Usage

<HCard title="🚀 Use HINATA" link="/en/usage" cols="1">
  HINATA can be used in many ways, not only as a game card reader.
</HCard>

## Configure Reader Settings

<HCard title="🔧 Change HINATA Settings" link="configuration" cols="1">
  Use HINATA Go to change HINATA settings.
</HCard>

## Update Reader Firmware

<HCard title="⏫ Update HINATA Firmware" link="update/" cols="1">
  Use HINATA Go or HINATA Client to update HINATA firmware.
</HCard>

## HINATA Go

<HCard title="🌏 HINATA Go" link="go/" cols="1">
  HINATA Go can turn your device into a remote card reader for games. It can also read card information, manage cards, and connect to HINATA for configuration or updates.
</HCard>
